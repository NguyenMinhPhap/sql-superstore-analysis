# sql-superstore-analysis
Retail Sales Analysis using SQL (Superstore Dataset)
## Project Overview
This project analyses deatail sales data by using SQL.
The dataset used is the Superstore dataset.
## dataset 
Source: Superstore Dataset  
Contains information about orders, customers, sales, profit, and regions.
##  Key Analysis Questions
1. What are the top selling product categories?
2. Which region generates the highest sales?
3. Which customers generate the most profit?
4. Monthly sales trends over time
